# Soft Atlas

A quiet collection of places, moments, and memories.

Soft Atlas is a personal travel journal built with React.  
It focuses on slower travel experiences, visual storytelling, and calm design.

## Features
- Responsive layout
- Image-based destination cards
- Smooth scroll interactions
- Clean typography and subtle transitions

## Tech Stack
- React
- React Router
- CSS

## Status
This project is actively evolving.
